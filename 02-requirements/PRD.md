## 1. 产品概述

### 1.1 产品名称
智能合同全生命周期管理系统（Intelligent Contract Lifecycle Management System, i-CLMS）

### 1.2 产品定位
以合同签署为自动化起点的企业级数据中枢，实现"一份合同签署，全流程数据就绪，全部门价值赋能"。

### 1.3 目标用户
- 财务部门
- PMO/交付部门
- 业务/销售部门
- 市场部门
- 法务/风控部门
- 管理层

## 2. 功能需求

### 2.1 核心引擎：智能解析与数据中枢

#### F2.1.1 多格式文档解析
- 支持Word (.docx)文档解析
- 支持PDF文档解析（含文字型PDF）
- 支持扫描件OCR识别
- 准确率目标：>85%

#### F2.1.2 合同类型自动识别
- 人力框架合同
- 项目外包合同
- 产品购销合同

#### F2.1.3 关键字段提取

##### 一、通用核心字段（所有合同类型必备）

| 字段类别 | 具体字段 | 说明 |
|----------|----------|------|
| **身份标识** | 合同编号 | 内部唯一编号，通常包含年份、客户/项目代码 |
| | 合同名称 | 如"XX公司2024年度IT技术支持服务合同" |
| **签约方信息** | 我方主体 | 签约的子公司或法律实体 |
| | 客户名称 | 客户公司全称 |
| | 客户统一社会信用代码 | 用于唯一标识客户 |
| **财务条款** | 合同总金额（含税） | 人民币总额 |
| | 合同总金额（不含税） | 用于财务计算 |
| | 币种 | 人民币、美元等 |
| | 税率与税额 | 增值税率及具体税额 |
| | 付款方式 | 如：月结、里程碑付款、货到付款等 |
| | 付款条件 | 具体的触发付款的事件或凭证 |
| **时间周期** | 合同签署日期 | 双方签字盖章生效的日期 |
| | 合同生效日期 | 开始执行的日期（可能与签署日不同） |
| | 合同终止日期 | 合同结束的日期 |
| | 合同期限 | 如"1年"、"自生效日起至项目验收完成" |
| **物理属性** | 合同文本存储路径 | 电子文档在系统中的位置或链接 |
| | 合同份数 | 正本、副本数量 |
| | 签订地点 | |
| **状态与分类** | 合同状态 | 草拟、审批中、已生效、执行中、已完结、已终止、已过期 |
| | 合同类型 | 人力框架、项目外包、产品购销 |
| | 所属行业 | 金融、制造、教育等，用于市场分析 |
| | 业务部门/销售负责人 | 内部责任主体 |

##### 二、类型特定字段

**1. 人力框架合同（基于工时计费）**

核心逻辑：`总费用 = Σ(人员级别 × 工时 × 费率)`

| 字段类别 | 具体字段 | 说明 |
|----------|----------|------|
| **资源与费率** | 人员级别/角色 | 如：资深架构师、项目经理、开发工程师 |
| | 对应费率（时/日/月） | 每个级别的标准单价 |
| | 费率有效期 | 价格可能每年调整 |
| **工作量与上限** | 预计总工时/人日 | 合同预估的总工作量 |
| | 每月/每年工时上限 | 控制成本的风险条款 |
| | 工时填报与审核流程 | 如何确认实际工作量 |
| **结算与调整** | 结算周期 | 按月、按季度结算 |
| | 工时调整机制 | 超出部分的处理方式 |
| **特殊条款** | 人员更换流程 | 客户对资源的审批权 |

**2. 项目外包合同（基于里程碑）**

核心逻辑：`总费用 = 里程碑1金额 + 里程碑2金额 + ...`

| 字段类别 | 具体字段 | 说明 |
|----------|----------|------|
| **交付范围** | 工作说明书（SOW）范围摘要 | 项目目标的简要描述，是解析重点 |
| | 关键交付物清单 | 需要提交的文档、代码、报告等 |
| **里程碑计划** | 里程碑名称与顺序 | 如：需求确认、系统设计、上线验收 |
| | 各里程碑交付物 | 每个里程碑对应的具体产出 |
| | 各里程碑对应金额/付款比例 | 如：首付30%，上线付40%，终验付30% |
| | 各里程碑计划完成日期 | 合同约定的时间点 |
| **验收标准** | 里程碑验收标准 | 如何判定该里程碑完成 |
| | 最终验收标准与流程 | 项目整体关闭的条件 |
| **变更管理** | 范围变更流程 | 如何处理需求增减 |

**3. 产品购销合同**

核心逻辑：`总费用 = Σ(产品单价 × 数量) + 服务费`

| 字段类别 | 具体字段 | 说明 |
|----------|----------|------|
| **标的物明细** | 产品/服务清单 | 具体的产品名称、型号、软件许可名称 |
| | 规格与配置 | 产品的详细技术参数 |
| | 数量 | 采购数量 |
| | 单价（含税/不含税） | 每个产品的价格 |
| | 小计金额 | 单价×数量 |
| **交付与物流** | 交付内容 | 软件许可密钥、硬件设备、文档 |
| | 交付日期 | 明确的交货时间点或期限 |
| | 交付地点 | 客户指定地点 |
| | 运输与保险责任 | 费用和风险由谁承担 |
| **产权与售后** | 知识产权归属 | 特别是定制化开发部分 |
| | 保修期 | 硬件保修、软件维保的起止时间 |
| | 售后服务条款 | 响应时间、服务等级协议（SLA） |

## 3. 非功能需求

### 3.1 性能
- 页面加载时间 < 2秒
- 合同解析时间 < 30秒
- 支持10并发用户

### 3.2 安全
- HTTPS传输加密
- 数据库加密存储
- RBAC权限控制

### 3.3 可用性
- 系统可用性 > 99.5%
- 数据每日备份
